=====================================================================
TECHNICAL REQUIREMENTS
=====================================================================
- Vite + React 18 (functional components + hooks)
- TailwindCSS for all styling (no separate CSS files)
- axios or native fetch for GitHub API calls
- useEffect + useState for async data; show skeleton loaders while loading
- Handle GitHub API errors gracefully (fall back to static data above)
- Mobile-first, fully responsive
- Dark mode toggle (persisted in localStorage)
- Smooth scroll-spy nav that highlights the active section
- No TypeScript required — plain JS is fine
- README with: project description, setup steps (`npm install && npm run dev`),
  and a note about the optional VITE_GITHUB_TOKEN env var to raise API rate limit
